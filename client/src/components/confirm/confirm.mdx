# Confirm Component

Global component ï¼Œthat provides secondary validation and retrieves results through callback.
When multiple simultaneous calls occur, the last content is shown first.

## Code Usage

```tsx
import { ConfirmContext } from '@client/components/confirm/context';

export const Example = () => {
    const { showConfirm } = useContext(ConfirmContext);

    const handleShowConfirm = () => {
        showConfirm('confirm message', (confirm) => {
            if (confirm) {
                console.log('confirmed');
            } else {
                console.log('canceled');
            }
        });
    };
};
```
